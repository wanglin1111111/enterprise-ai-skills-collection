### 案例1：学习者画像四维诊断

**场景**：初中数学学习者个性化诊断

**输入**：
```
学习者：初中二年级学生
学习数据：
- 二次函数题目错误率60%
- 答题时间长（平均5分钟/题）
- 观看视频时长占比70%
- 兴趣点：篮球、游戏
```

**预期分析路径**：
1. ✅ 知识状态诊断：根源薄弱点是"函数概念理解不足"（非二次函数本身）
2. ✅ 认知风格识别：视觉型学习者（视频偏好70%）
3. ✅ 学习动机分析：内在动机强（兴趣点：篮球、游戏）
4. ✅ 情感状态监测：轻度焦虑（答题时间长、错误率高）

**验证点**：
- ✅ 知识薄弱点定位准确（根源识别）
- ✅ 认知风格识别正确（视觉型）
- ✅ 动机分析合理（内在动机）
- ✅ 情感状态判断准确（轻度焦虑）

**评分标准**：
- 知识诊断：25分
- 风格识别：25分
- 动机分析：25分
- 情感判断：25分

**预期得分**：92/100 (A)

**干预方案**：
1. 先复习函数基础概念（视频形式）
2. 二次函数题目融入篮球场景（兴趣关联）
3. 分解复杂题目为简单步骤（缓解焦虑）
4. 提供图表和视频讲解（适配视觉型）

### 案例2：AI教学内容生成

**场景**：为阅读型学习者生成供需曲线解释

**输入**：
```
概念：供需曲线
认知风格：阅读型（偏好文本）
难度：入门级
兴趣点：经济学应用
```

**预期生成内容**：
```
标题：供需曲线：市场价格的幕后推手

正文：
想象一个苹果市场，这里每天都在上演着供需的博弈。

【供给曲线：卖家的心理】
苹果价格越高，果农愿意卖得越多。
这就像你考试考得越好，越愿意展示成绩一样。
供给曲线向上倾斜，反映了卖家的逐利心理。

【需求曲线：买家的选择】
苹果价格越高，消费者买得越少。
这就像你看到商品打折时，忍不住多买几件一样。
需求曲线向下倾斜，反映了买家对价格的敏感性。

【市场均衡：价格的魔法】
当供给和需求两条曲线相交时，就形成了市场均衡价格。
这就像天平找到了平衡点，买卖双方都满意。

关键要点：
✓ 供给曲线向上：价格↑，供给↑
✓ 需求曲线向下：价格↑，需求↓
✓ 均衡价格：供给=需求时的价格

思考题：如果苹果丰收，供给曲线会如何移动？市场价格会如何变化？
```

**验证点**：
- ✅ 文本为主（适配阅读型）
- ✅ 生活化类比（苹果市场）
- ✅ 结构清晰（标题、正文、要点）
- ✅ 互动元素（思考题）

**评分标准**：
- 风格适配：25分
- 内容准确性：25分
- 易理解性：25分
- 互动设计：25分

**预期得分**：90/100 (A)

### 案例3：学习路径动态优化

**场景**：成人Python数据分析学习路径优化

**输入**：
```
学习者：成人零基础
背景：有Excel数据处理经验
目标：掌握Python数据分析
传统路径：Python基础(4周) → 数据结构(2周) → 数据分析库(2周) = 8周
```

**预期优化路径**：
1. ✅ 第1周快速评估：学习者Excel经验表明数据处理能力已具备
2. ✅ 跳过策略：跳过数据结构基础，直接进入Pandas库学习
3. ✅ 补充机制：遇到列表概念时，补充Python基础相关知识
4. ✅ 动态调整：根据学习效果调整难度和节奏

**验证点**：
- ✅ 背景能力识别准确（Excel经验对应数据处理能力）
- ✅ 跳过决策合理（跳过已掌握内容）
- ✅ 补充机制完善（按需补充先修知识）
- ✅ 效率提升显著（8周→5周，效率提升37.5%）

**评分标准**：
- 能力识别：25分
- 路径优化：30分
- 补充机制：20分
- 效率提升：25分

**预期得分**：95/100 (A)

**场景**：云南山区小学英语教育提升

**输入**：
```

目标：提升至全县中游水平

```

- 每月评估：听写测试、口语测试
- 动态调整：根据成绩调整教学节奏

- 2年：成绩提升至全县中游
- 3年：成绩稳定在全县中游水平

**验证点**：
- ✅ 方案可行（AI教师 + 本地教师）
- ✅ 职责分工明确（技术vs情感）
- ✅ 效果评估完善（每月评估、动态调整）

**评分标准**：
