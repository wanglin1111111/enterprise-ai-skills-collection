# AI-Driven Economic Research - Test Cases

## 测试案例集

### 案例1：政策文本分析

**场景**：分析中央银行货币政策对市场的影响

**输入**：
```
数据源：中国人民银行货币政策执行报告（2020-2026年）
目标：构建货币政策不确定性指数，预测市场反应
```

**预期分析路径**：
1. 数据准备：收集货币政策报告文本，清洗格式
2. 特征提取：使用BERTopic提取主题，VADER进行情感分析
3. 计量分析：构建不确定性指数，与股市波动率做相关性分析
4. 预测建模：使用LSTM模型预测未来市场波动

**验证点**：
- ✅ 主题提取合理性（货币政策、流动性、风险等主题）
- ✅ 情感分析准确性（鹰派/鸽派信号识别）
- ✅ 不确定性指数与市场波动相关性（预期r>0.5）
- ✅ 预测模型准确性（预期RMSE<10%）

**评分标准**：
- 主题提取合理：20分
- 情感分析准确：20分
- 相关性显著：30分
- 预测准确性：30分

**预期得分**：85-95/100

### 案例2：卫星图像经济评估

**场景**：使用卫星图像评估长三角区域经济发展

**输入**：
```
数据源：Google Earth卫星图像（2015-2025年，长三角区域）
目标：估计各城市GDP，验证统计数据准确性
```

**预期分析路径**：
1. 图像采集：下载长三角城市卫星图像（时间序列）
2. 特征提取：识别建筑物密度、道路网络、车辆数量
3. 经济建模：建立图像特征与GDP的回归模型
4. 验证分析：与官方GDP数据对比，识别异常区域

**验证点**：
- ✅ 建筑物识别准确率（预期>90%）
- ✅ 道路网络提取完整性（预期>85%）
- ✅ GDP估计相关性（预期r>0.8）
- ✅ 异常区域识别（发现统计数据矛盾）

**评分标准**：
- 特征提取准确性：30分
- 经济建模合理性：30分
- 验证分析有效性：20分
- 异常识别价值：20分

**预期得分**：80-90/100

### 案例3：AI就业影响评估

**场景**：评估AI对制造业就业的影响

**输入**：
```
数据源：中国制造业岗位数据、技能数据、自动化技术数据
目标：预测2025-2030年制造业就业结构变化
```

**预期分析路径**：
1. 岗位分类：按自动化风险分为高/中/低风险
2. 替代模型：基于技术发展预测替代率
3. 创造模型：预测新岗位创造率
4. 政策建议：提出教育转型和社会保障建议

**验证点**：
- ✅ 岗位分类合理性（高风险：流水线工人，低风险：研发工程师）
- ✅ 替代率预测合理性（预期20-40%）
- ✅ 创造率预测合理性（预期10-20%）
- ✅ 政策建议针对性（技能培训、终身学习、失业保险）

**评分标准**：
- 岗位分类准确：25分
- 替代模型合理：25分
- 创造模型合理：25分
- 政策建议可行：25分

**预期得分**：75-85/100

### 案例4：健康经济学研究

**场景**：研究糖尿病对患者消费行为的影响

**输入**：
```
数据源：超市购物数据、医疗诊断数据、患者调查数据
目标：分析糖尿病确诊前后的消费行为变化
```

**预期分析路径**：
1. 患者识别：用机器学习从购物数据识别潜在糖尿病患者
2. 行为分析：对比确诊前后的消费模式变化
3. 经济负担：计算疾病导致的直接和间接经济损失
4. 政策建议：提出慢性病管理策略

**验证点**：
- ✅ 患者识别准确率（预期>80%）
- ✅ 行为变化显著（确诊后健康食品购买增加）
- ✅ 经济负担量化合理性
- ✅ 政策建议针对性（早期筛查、健康教育）

**评分标准**：
- 患者识别准确：25分
- 行为分析深入：25分
- 经济负担量化：25分
- 政策建议可行：25分

**预期得分**：80-90/100

### 案例5：跨学科融合 - 环境经济学

**场景**：评估空气质量对房地产价格的影响

**输入**：
```
数据源：空气质量监测数据、房地产交易数据、街景图像
目标：量化空气质量改善对房价的因果效应
```

**预期分析路径**：
1. 数据整合：空气质量数据 + 房价数据 + 街景图像
2. 因果识别：双重差分法 + 断点回归（政策冲击）
3. 特征提取：街景图像识别绿化率、建筑物密度
4. 政策评估：空气质量改善的经济价值

**验证点**：
- ✅ 因果识别策略合理（政策冲击作为外生变量）
- ✅ 特征提取准确性（绿化率识别>85%）
- ✅ 因果效应显著（空气质量改善→房价上涨）
- ✅ 经济价值量化合理性

**评分标准**：
- 因果识别合理：30分
- 特征提取准确：20分
- 因果效应显著：30分
- 经济价值量化：20分

**预期得分**：85-95/100
