# AI极限能力激活方法论 - Quick Reference

## 核心框架速查

```
能力认知层 → 激活策略层 → 应用场景层 → 影响评估层
（基准/Pro优势/边界）→（对话/理论/创造力）→（科研/创作/推理/哲学）→（思维改变/思想家催生）
```

## 三大激活策略速查

### 策略1：深度对话三层递进法

```
初始触发 → 连续追问 → 洞见提取
（核心问题）→（抓住话头）→（深层认知）
```

**关键操作**：
- 识别第一轮回答的"话头"
- 基于话头继续追问（至少3轮）
- 从追问链中提取深层洞见

**案例**：
```
潜艇vs空间站 → 空间站长（表面）→ 补给方式差异（根本）
常规模型：表面答案
深度对话：根本原因（可持续性 vs 有限性）
```

---

### 策略2：理论创新激发框架

**四步骤流程**：
```
领域边界设定 → 理论激发提问 → 实验方法设计 → 理论评估
（研究领域）→（新理论构想）→（验证方法）→（创新性评估）
```

**提问模板**：
- "基于现有理论X，是否可以构想新理论Y？"
- "如何验证这个新理论？"
- "实验设计步骤是什么？"

**关键原则**：
- ✅ 不提供任何提示或帮助
- ✅ 要求AI自主提出完整理论框架
- ✅ 强调科学严谨性和可实施性

---

### 策略3：极限创造力释放策略

**三层次模型**：
```
要求设定 → 自主创作 → 创作评估
（不给帮助）→（AI自主）→（深度评估）
```

**关键原则**：
- ✅ 只提要求，不给素材、提示、帮助
- ✅ AI完全自主选择主题、结构、风格
- ✅ 人类仅作为"要求提出者"

**案例**：
```
微小说创作要求：
- 主题：超级AI时代普通人精神挣扎
- 需包含：人性、秩序、不确定性
- 不给：素材、提示、帮助
结果：AI自主创作《零序思块》+ 深层哲学思考
```

## 五大核心原则

1. **难题优先** - 常规用主流，难题用Pro
2. **连续追问** - 抓住话头，至少3轮
3. **不给帮助** - 理论激发和创作时只提要求
4. **能力边界** - 识别常规 vs 高难度差距
5. **洞见提取** - 提取超出预期的深层认知

## 典型使用模式

### 模式1：深度对话激活
```
用户: 核心问题
AI: 第一轮回答（识别话头）
用户: 基于话头追问
AI: 第二轮回答（识别话头）
用户: 基于话头追问
AI: 第三轮回答（深层洞见）
```

### 模式2：理论激发激活
```
用户: 领域背景 + 理论激发提问（不给帮助）
AI: 新理论框架 + 假设
用户: 要求实验方法设计（不给帮助）
AI: 实验设计 + 验证步骤
用户: 评估理论创新性
```

### 模式3：创造力释放激活
```
用户: 创作要求（不给素材、提示、帮助）
AI: 自主创作（主题、结构、风格自主选择）
用户: 评估创作深度和原创性
```
