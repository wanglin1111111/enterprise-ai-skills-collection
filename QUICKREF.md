# AI-Driven Economic Research - Quick Reference

## 核心框架速查

```
数据层 → 方法层 → 应用层 → 决策层
（文本/图像/传统）→（NLP/CV/ML）→（政策/预测/社会/跨学科）→（建议/投资/风险）
```

## 四大方法论速查

### 1. 文本数据量化分析三步法
```
数据准备 → 特征提取 → 计量分析
（清洗标注）→（主题/情感/实体）→（时间序列/相关性/因果）
```

**工具**：
- 主题建模：LDA、BERTopic
- 情感分析：TextBlob、VADER、BERT
- 实体识别：spaCy、Stanford NER

---

### 2. 图像数据经济分析四维度

| 维度 | 图像类型 | 提取信息 | 经济应用 |
|------|---------|---------|---------|
| 固定资产 | 卫星图像 | 建筑物、道路、车辆 | 经济总量评估 |
| 环境质量 | 街景图像 | 树木、绿地、污染 | 生活质量评估 |
| 社会行为 | 街景图像 | 行人、活动、涂鸦 | 社会活力评估 |
| 个体特征 | 医疗影像 | 健康指标、疾病 | 健康经济学 |

**工具**：
- 图像识别：TensorFlow、PyTorch、YOLO
- 变化检测：OpenCV、GDAL
- 经济建模：GeoDa、ArcGIS

---

### 3. 预测与因果推断融合策略

| 策略 | 流程 | 适用场景 |
|------|------|---------|
| 双模型法 | ML预测 → 计量解释 | 经济增长预测 |
| 特征工程辅助 | ML提取特征 → 计量建模 | 政策效果评估 |
| 因果机器学习 | Causal Forest、Double ML | 异质性分析 |

**工具**：
- 机器学习：scikit-learn、XGBoost、LightGBM
- 计量经济学：statsmodels、linearmodels
- 因果推断：EconML、DoWhy、CausalML

---

### 4. 就业影响评估三层次模型

```
第一层：岗位替代风险
（高风险/中风险/低风险）

第二层：新岗位创造
（直接/间接/潜在）

第三层：政策应对
（教育转型/社会保障/产业政策）
```

**关键指标**：
- 替代率 = AI替代岗位占比
- 创造率 = AI新创造岗位占比
- 净就业影响 = 创造率 - 替代率

## 案例速查

- **方法**：主题建模 + 情感分析
- **应用**：预测市场波动

## 五大核心原则

1. **方法论融合**：AI补充而非替代传统计量经济学
2. **数据驱动**：非传统数据与传统数据结合
3. **预测与因果并重**：追求准确性不放弃解释性
4. **伦理优先**：数据隐私、算法偏见、政策影响评估
5. **跨学科协作**：经济学 + 计算机科学 + 统计学 + 领域专家
